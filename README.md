## Why this matters
GNSS accuracy hinges on how well we can predict clock and orbital errors. Even subtle, time-varying deviations can degrade navigation and precision timing. This repository prototypes AI-driven solutions to detect and forecast these error patterns so operators and downstream systems can act earlier and more effectively.
